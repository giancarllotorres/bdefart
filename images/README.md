# Image Setup Instructions

Please save your images to the following locations:

1. **Profile Picture (276.png)**
   - Save as: `images/profile.png`
   - This circular avatar will appear in the About section

2. **Banner/Header Image (Bea Barbosa banner)**
   - Save as: `images/banner.png`
   - This will appear at the top of the hero section

## How to Save:

1. Right-click on each attached image in VS Code
2. Select "Save As..."
3. Navigate to the `images` folder
4. Use the filenames specified above

The website is already configured to use these images once they're in place!
